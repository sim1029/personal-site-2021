FEATURED PROJECT

PizzaBuddy

The first ever navigation tool created for and by pizza drivers. Modern maps and navigational tools provide uncessary features and do not track vital metrics that pizza drivers require at a glance throughout a shift. Time is money!

Pizza buddy solves all these problems, starting with a minimalistic approach to directions. Drivers know their area well and only require street names to find a location. This allows for faster routing despite slow service and adding multiple stops to find the best route from house to house. Keep track of lifetime earning and shift metrics all within the app. As more drivers enter their tips, more reliable tip ratings for houses appear.

This app was never released on the appstore but I used it during my own shifts as a pizza driver and it worked great! I will hopefully find time to release it soon as it supports user auth, data storing, and realtime directions.

PizzaBuddy: Driver's assistant

Navigation tool: Swift, Firebase, RealmSwift, Mapkit, Xcode

FEATURED PROJECT

Flight-Saver

Flying accross the country shouldn't cost an arm and a leg, and it doesn't have to anymore!

Introducing flight-saver, an intuitive and powerful tool to find the cheapest flights to your desired destination. Flight-saver grabs data from the skyskanner api and uses React to filter the data for you without having to query the API again.

The secret to this web-app's dazzling UI is Grommet, a react component library.

Sadly, the skyskanner API no longer offers free requests, and so the website is no longer working. However, feel free to checkout what I made on my public github repo!

Flight-Saver: Connect travelers with cheap flights

Travel Web App: React, APIs, Express, Grommet
